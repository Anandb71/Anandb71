## Operating System

I build production-grade developer tooling for autonomous AI systems: code intelligence, agent observability, local-first desktops, and interfaces that make complex systems feel inspectable instead of mysterious.

My current north star is simple: if an AI agent changes code, routes context, calls a tool, or touches a workflow, the system should be able to explain what happened, why it happened, and what might break next.

  <tr>

## Build Bias

```text
deterministic core > explainable state > fast feedback > cinematic interface
```

- I prefer graphs, traces, and contracts over vague context blobs.
- I ship local-first where privacy, latency, or trust matters.
- I like tools that answer "what changed?", "what depends on it?", and "why is it happening?"
- I design UI like the system has a pulse, but the internals still have to be boringly reliable.
